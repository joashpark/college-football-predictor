# College Football Prediction Model

## Overview
This project builds a probabilistic model to predict NCAA football game outcomes using historical data from 2002–2026. The model combines Elo-based ratings, margin-of-victory weighting, and logistic probability functions to estimate win probabilities.

---

## Methods Used
- Elo rating system
- Logistic probability function
- Margin-of-victory weighting
- Conference strength analysis
- Monte Carlo simulation (planned extension)

---

## Data
- NCAA football game data (2002–2026)
- Features include:
  - team names
  - scores
  - rankings
  - conference information
  - home/away status

---

## Key Features
- Team strength rating system
- Win probability prediction
- Home field advantage modeling
- Upset detection
- Team ranking system

---

## Results
The model achieves predictive accuracy above a baseline random model and captures meaningful differences in team strength across seasons and conferences.
